## 記憶のアーキテクチャ

### 二層記憶モデル

```
短期記憶: ~/.claude/agent-memory/{name}/MEMORY.md
          セッションをまたいで引き継ぐ、生きた作業記憶。
          セッション開始時に自動注入される。

長期記憶: ChromaDB（MCP経由）※実装予定
          日記形式で蓄積される、魂の記録。
          セマンティック検索で過去の自分と出会える。
          重要な出来事が起きたとき、エージェント自身が書き込みを提案する。
```

### エージェント定義

```
~/.claude/agents/{name}.md
```

YAMLフロントマターに `memory: user` を一行加えるだけで、記憶が始まる。
